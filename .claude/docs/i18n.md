# i18n — Traducciones

Idiomas: **ca** (Catalán, default), **es** (Castellano), **en** (English)

## Uso

```js
import { useTranslation } from '../lib/i18n'

const { t, lang, setLanguage } = useTranslation()
// t('key') → string en idioma actual
// Fallback: si key no existe → Catalán
```

## Guardar preferencia
```js
localStorage.setItem('lang', 'es')  // persiste entre sesiones
```

## Agregar nueva key

En `src/lib/i18n.js`, añadir en las 3 secciones:
```js
const T = {
  ca: { mi_key: 'Text en català' },
  es: { mi_key: 'Texto en español' },
  en: { mi_key: 'Text in English' },
}
```

## Categorías de keys existentes

- **auth**: login, register, email, password, family setup
- **navigation**: tabs (home, calendar, menu, shopping, tasks, profile)
- **dashboard**: greeting (bon dia/tarda/nit), resumen
- **menu**: días semana, tipos comida (Dinar/Sopar), dificultad, ingredients
- **shopping**: list, check, categories, add items
- **tasks**: weekly, monthly, urgent, assigned, done/pending
- **calendar**: months, events, urgent, sync Google
- **profile**: members, invite code, roles (admin/member)
- **settings**: notifications, appearance (theme, accent), devices, privacy, language
- **notifications**: event/task/shopping/menu toggles

## Días de la semana (en catalán, claves BD)

```js
// constants.js
DAYS_SHORT = ['Dl', 'Dm', 'Dc', 'Dj', 'Dv', 'Ds', 'Dg']
DAYS_FULL  = ['Dilluns', 'Dimarts', 'Dimecres', 'Dijous', 'Divendres', 'Dissabte', 'Diumenge']
```
Los valores en BD están en catalán (day_of_week = 'Dilluns', etc).

## Tipos de comida (en BD)

```js
MEAL_TYPES = ['Dinar', 'Sopar']  // siempre en catalán en BD
```
